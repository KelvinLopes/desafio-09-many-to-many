Aqui se pode simular o cadastro de cliente, produtos, emissão de ordens que atualizam a quantidade ao produto ser "vendido".

Essas ordens, terão o cliente, produtos e quantidades compradas de cada um por esse cliente.

### Como usar?

Baixe o repositório na opção de download ou :

Crie um diretório via terminal ou via gerenciador de arquivos.

No seu terminal acesse essa pasta e execute o comando abaixo:

git clone https://github.com/KelvinLopes/desafio-09-many-to-many

Após ter acesso ao diretório do projeto rode esse comando para instalar todas as dependências:

```yarn```

Caso necessário rode também:

```yarn upgrade```

Rodar o projeto ```yarn dev:server```

Rodar os testes no projeto ```yarn test```

### Antes de rodar os testes lembre se de criar a variável ambiente em arquivo .env .

Você pode precisar para testar a api algum Client Rest como o Insomnia.
